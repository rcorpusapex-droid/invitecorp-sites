# Plantilla Invitación Web — Clásica con Portada a Pantalla Completa

Esta es una **plantilla original** (no es el código del sitio de terceros).
La estructura y el look del HERO (foto fullscreen con texto centrado) están pensados para verse como tu captura.

## Editar textos
Todo se cambia en `config.js`.

## Cambiar portada y fotos
- Reemplaza `assets/hero.jpg` por tu foto principal (mismo nombre o actualiza `config.js`).
- Reemplaza `assets/photo-1.jpg` ... `assets/photo-8.jpg` por tus fotos.

## Publicar
Sube la carpeta a Netlify / Vercel / GitHub Pages.

## RSVP real
El formulario es demo. Si quieres, se conecta a Google Sheets / Airtable / Supabase.
